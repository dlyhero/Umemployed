Question: Sorting Algorithm

Welcome to the sorting challenge! Your task is to implement a sorting algorithm that can sort a list of integers in ascending order. Follow the instructions below to complete the challenge:

  1. Write a function named sortList that takes a list of integers as input.
  2. Inside the function, implement a sorting algorithm of your choice (e.g., bubble sort, selection sort, insertion sort, etc.) to sort the list in ascending order.
  3. The function should return the sorted list.

Example:
If the input list is [5, 2, 9, 1, 3], the function should return [1, 2, 3, 5, 9].
